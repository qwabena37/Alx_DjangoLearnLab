# Django Blog Project

## Overview

This project is a comprehensive blogging platform built with Django. It features user authentication (registration, login, logout), profile management, and basic blog post functionality.

---

## Features

- **User Authentication**
  - User registration with email and password
  - Login and logout using Djangoâ€™s built-in auth system
  - Profile page for viewing and updating user information

- **Blog Posts**
  - Create and manage blog posts with title, content, author, and publish date

- **Responsive UI**
  - HTML templates styled with CSS (customizable)

- **Database Support**
  - Default SQLite support
  - Optional PostgreSQL configuration for production
